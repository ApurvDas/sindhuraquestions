# 🎬 Movie Night Invite 💜

A playful, mobile-first single-page web app built to ask my girlfriend, Sindhura, to watch a movie together. Features a lavender-themed UI, interactive animations, a live countdown, and a Supabase backend to securely save her response. Built with love (and a gentle reminder to be on time 😄).

**Tech:** `HTML5` • `CSS3` • `Vanilla JS` • `Supabase` • `GitHub Pages`
